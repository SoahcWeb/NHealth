# Projet NHealth

NHealth est un module Laravel de suivi santé et transformation personnelle.

Technologies :
- Laravel
- Blade
- Tailwind CSS
- Chart.js
- SQLite

Règles :
- Ne jamais supprimer du code existant sans demander.
- Toujours expliquer les fichiers modifiés.
- Toutes les données doivent appartenir à user_id.
- Toutes les routes doivent être protégées par auth.
- Utiliser les conventions Laravel.
- Garder un code propre, maintenable et commenté.
- Travailler étape par étape.
